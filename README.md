# Scandinavian SSN Generator and Validator

This project allows you to generate and validate Scandinavian Social Security Numbers (SSNs) for Sweden, Norway, Finland, Iceland, and Denmark. The generator creates valid SSNs based on the specific formats used in each country, while the validator checks if a given SSN is valid according to the respective country's rules.

## Features
- Generate valid SSNs for Sweden, Norway, Finland, Iceland, and Denmark.
- Validate SSNs for the same countries.
- Easy-to-use
- All processes are done on the client side, ensuring privacy and security.
